Кейс1
Шаг 1: Перейти в корзину
Ожидаемый результат:
Пользователю отображается шторка с корректным содержимым, включающим следующие элементы:
1.	Заголовок с адресом и выпадающим списком:
•	Отображается заголовок с текущим адресом доставки.
•	Возле адреса присутствует выпадающий список для выбора другого адреса.
2.	Комментарий со стрелкой вправо:
•	Поле "Комментарий" доступно для редактирования.
•	Справа от поля отображается стрелка, указывающая направление вправо.
3.	Тогл "Оставить у двери":
•	Присутствует тогл-переключатель с текстом "Оставить у двери".
•	Тогл находится в выключенном состоянии по умолчанию.
4.	Заголовок "Доставка" с временным форматом (XX минут):
5.	Список продуктов:
•	Отображается список всех продуктов, добавленных в корзину.
•	Для каждого продукта указаны название, цена, количество и общая стоимость.
6.	Меню "Скидка или промокод":
•	Доступно меню для ввода скидки или промокода.
•	При нажатии на меню открывается поле для ввода данных.
7.	Сбер айди:
•	Отображается информация о подключении через Сбер айди.
•	Присутствует возможность авторизации через Сбер айди.
8.	Меню "Оплата картой":
•	Отображается меню с выбранным способом оплаты "Оплата картой".
•	Возможность изменить способ оплаты через выпадающий список.
9.	Итоговая сумма:
•	Отображается итоговая сумма заказа.
•	Сумма рассчитана корректно, включая все товары, скидки и доставку.
10.	Кнопка "Оплатить":
•	Кнопка "Оплатить" активна и доступна для нажатия.
•	На кнопке отображается текст "Оплатить" и сумма к оплате.
11.	Дополнительное меню "Добавить к заказу":
•	В нижней части шторки доступно дополнительное меню "Добавить

Кейс2
Предусловие:
1. Зайти в приложение "Самокат"
2. Выбрать один продукт
3. У пользователя добавлено несколько адрес для доставки

Шаг 1
Перейти в корзину
ОР:Пользователю отображается шторка с выбранным товаром

Шаг 2
Тапнуть по заголовку с адресом
ОР:Пользователю отображается шторка со списком ранее добавленных адресов. Шторка содержит следующие элементы:
   1. Заголовок «Выберите адрес»:
   2. Слева от каждого адреса отображается чекбокс. Чекбокс выбранного адреса выделен розовым цветом, остальные чекбоксы находятся в неактивном состоянии.
   3. Название улицы, номер дома и квартиры:
   •	Для каждого адреса отображается полное название улицы, номер дома и номер квартиры (если указан).
   4. Троеточие справа от адреса
   5. В нижней части экрана отображается кнопка «Новый адрес».

Шаг 3
Тапнуть по неактивному чекбоксу
ОР: Выполнен переход на экран корзины
    В заголовке адреса указан выбранный новый адрес

Шаг 4
Тапнуть по заголовку с адресом
ОР: Чек бокс с новым выбранным адресом активен

Кейс3
Добавление адреса в корзине
Предусловие:
1. Зайти в приложение "Самокат"
2. Выбрать один продукт
3. У пользователя добавлено несколько адрес для доставки

Шаг 1: Перейти в корзину
ОР: Пользователю отображается шторка с выбранным товаром

Шаг 2: Тапнуть по заголовку с адресом
ОР: Выполнен переход на шторки  «Выберите адрес»

Шаг 3: Тапнуть по кнопка «Новый адрес»
Ор: Пользователю отображается google карта
    Внизу карты отображаются элементы:
    1. Заголовок название адреса
    Кнопки:
    1. Да, всё верное
    2. Ввести другой адрес

Шаг 4: Выбрать на карте любой адрес
ОР: Адрес отображается в заголовке

Шаг 5: Тапнуть по кнопке «Да, все верно»
Ор: Выполнен переход на экран корзины
    Вверху заголовка отображается созданный ранее адрес

Кейс4
Удаление адреса

Предусловие:
1. Зайти в приложение "Самокат"
2. Выбрать один продукт
3. У пользователя добавлено несколько адрес для доставки

Шаг 1: Перейти в корзину
ОР: Пользователю отображается шторка с выбранным товаром

Шаг 2: Тапнуть по заголовку с адресом
ОР: Выполнен переход на шторку  «Выберите адрес»

Шаг 3: Тапнуть по троеточию напротив адреса
ОР: Пользователю отображается экран с подробным описанием адреса и следующими элементами:
    1. Кнопка слева «Назад»
    2. Кнопка справа «Удалить адрес»
    3. Заголовок название улицы и номер дома
    Строки:
    1. Квартира или офис
    2. Подъезд
    3. Этаж
    4. Код домофона
    5. Комментарий
    Внизу экрана кнопка «Сохранить»

Шаг 4: Тапнуть по кнопке «Удалить адрес»
ОР: Пользователю отображается окно «Удалить адрес?»
    Кнопки:
    Да (выбрано по умолчанию)
    Нет

Шаг 5: Тапнуть на кнопку «Да»
ОР: Выполнен переход на экран «Выберите адрес»
    Адрес успешно удален
